> A popover allows the user to reveal more information or non-critical tasks.

## Anatomy

1. **Container** - the popover's container overlays the main page's content.
1. **Arrow** - a visual indicator that points to the related element.
1. **Title** (_recommended_) - the title of a callout summarizes its contents.
1. **Close button** (_optional_) - a button that will always close the popover.
1. **Body** - the contents of the popover.
1. **Action buttons** (_optional_) - one or more custom actions that will result in the popover closing.

## Usage

Use a popover for progressive disclosure of information or actions that don't belong on the main page.

<Usage
	bad="Open a popover on focus."
	good="Open a popover by clicking a button."
>

Popovers must be focused when they open, which means that opening one by focusing the trigger element would cause focus to move twice in one action.
This is a violation of [WCAG 3.2.1: On Focus](https://www.w3.org/WAI/WCAG21/Understanding/on-focus.html).

</Usage>

<Usage
	bad="Use a popover to label an element."
	good="Use a tooltip instead."
>

Even thought they look similar, popovers and tooltips serve two very different purposes.
Popovers are reserved for complex related content or tasks, whereas tooltips are used to name or describe a related element.

</Usage>


## React API

We expose one component for the popover API, the [Popover](https://github.com/wwnorton/design-system/tree/main/packages/react/src/components/Popover).

### Popover

The `Popover` component extends the `React.HTMLAttributes<HTMLElement>` interface and uses [Popper.js](https://popper.js.org/) for positioning.

```js
import { Popover } from '@wwnds/react';
```

```tsx live
function PopoverExample() {
	const [isOpen, setIsOpen] = React.useState(false);
	const [ref, setRef] = React.useState();

	return (
		<>
			<Button variant="solid" ref={setRef}>
				Show popover
			</Button>
			<Popover
				title="Title"
				hideTitle={false}
				hideCloseButton={false}
				placement="top"
				reference={ref}
				isOpen={isOpen}
				onRequestClose={() => setIsOpen(false)}
				onRequestOpen={() => setIsOpen(true)}
			>
				Lorem ipsum dolor sit amet, consectetur adipiscing elit,
				sed do eiusmod tempor incididunt ut labore et dolore magna aliqua.
				Ut enim ad minim veniam, quis nostrud exercitation ullamco
			</Popover>
		</>
	);
}
```

    Accessibility

    variables

    focus-ring-inner

    $focus-ring-inner: props.prop('body-bg') !default;

    Description

    The color of the inner focus ring ring. Change this to the background color if the focused element isn't sitting on an element with a background color of body-bg.

    focus-shadow

    $focus-shadow: (
	0 0 0 1px props.prop('focus-ring-inner'),
	0 0 0 3px props.prop('focus-ring'),
) !default;

    Description

    The default focus shadow has a 2px outer ring and an inner ring that matches the body's background color by default, giving the illusion of transparency.

    focus-shadow-collapse

    $focus-shadow-collapse: 0 0 0 1px props.prop('focus-ring') !default;

    Description

    A collapsed focus shadow that is meant to be used the focused element conveys focus through some other mechanism, such as an <input>'s caret.

    focus-ring

    @mixin focus-ring($current-shadows...) { ... }

    Description

    A box shadow focus indicator. The rendered focus ring is offset from the focused element by 1px of focus-ring-inner (body-bg by default).

    Parameters

    parameter Nameparameter Descriptionparameter Typeparameter Default value
    $current-shadows...

    Zero or more additional shadows to be applied in addition to the focus indicator.

    		List[]— none

    focus-ring-collapse

    @mixin focus-ring-collapse($current-shadows...) { ... }

    Description

    A collapsed shadow focus indicator. This is similar to the normal focus ring but does not have the focus-ring-inner offset. Instead, it changes the color of the focused element's border to match the focus ring, creating the effect of a 2px focus ring.

    Parameters

    		List[]— none

    family

    @function family($name, $ref-family, $grade-root: null) { ... }

    Description

    Create a new color family map that references an existing family map.

    Parameters

    parameter Nameparameter Descriptionparameter Typeparameter Default value
    $name

    The name of the family that you'd like to create.

    		String— none
    $ref-family

    The name of the family that you'd like to reference.

    		String— none
    $grade-root

    If provided, this will be the midpoint of the "lighter/darker" shade swatch.

    		Numbernull

    Example

    Minimal usage

    @debug family('foo', 'teal');
// (
//   10: var(--nds-teal-10),
//   20: var(--nds-teal-20),
//   ...30 - 80...
//   90: var(--nds-teal-90),
//   100: var(--nds-teal-100),
// )

    With $grade-root

    @debug family('foo', 'teal', 30);
// (
//   10: var(--nds-teal-10),
//   20: var(--nds-teal-20),
//   ...all other grades...
//   'lighter': var(--nds-foo-10),
//   'light': var(--nds-foo-20),
//   '#root': var(--nds-foo-30),
//   'dark': var(--nds-foo-40),
//   'darker': var(--nds-foo-50),
// )

    Requires

    • [function] prop
    • [function] value
    • [variable] css-property-fallback

    easing-standard

    $easing-standard: cubic-bezier(0.4, 0, 0.2, 1);

    Description

    Standard easing puts subtle attention at the end of an animation, by giving more time to deceleration than acceleration. It is the most common form of easing.

    Elements that begin and end at rest use standard easing. They speed up quickly and slow down gradually, in order to emphasize the end of the transition.
